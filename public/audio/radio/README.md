# Radio station assets

Optional licensed station loops can be placed here as:

- `station-01.ogg`
- `station-02.ogg`
- `station-03.ogg`
- `station-04.ogg`
- `station-05.ogg`
- `station-06.ogg`

The game deliberately falls back to procedural static and station-identification tones when a file is absent or cannot be decoded.
